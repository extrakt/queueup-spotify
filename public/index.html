FAQs

  • Do I need to create an account?

    Only if you want to create a playlist or vote on a track.

  • Why can't I stream from the desktop site?

    Spotify does not have web streaming support.

  • Why do I need Spotify Premium to stream a playlist?

    Spotify requires that mobile users have Premium to stream

  • I want to report a bug, offer a suggestion, or give feedback

    QueueUp is in the beta phase, so feedback is always welcome here.
